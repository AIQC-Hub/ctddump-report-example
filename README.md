# ctddump: CTD data summary reports

A static web site of CTD data summary reports, one page per
(region, product) unit, generated by the
[`ctddump`](https://github.com/AIQC-Hub/ctddump) pipeline's
`summary_site.sh` (built with [mdBook](https://rust-lang.github.io/mdBook/)).

This directory is the built site: open `index.html` locally, or publish
it as-is with GitHub Pages (a `.nojekyll` is included).

Regions in this build: arctic baltic mediterranean.

Generated on 2026-07-17.
